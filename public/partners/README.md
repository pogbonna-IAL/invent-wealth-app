# Partner Logos

This directory contains logos for partner organizations displayed on the homepage.

## Required Logo Files

Please add the following logo files to this directory:

1. **scuml-logo.png** - SCUML (Special Control Unit Against Money Laundering)
2. **cardinalstone-logo.png** - CardinalStone Trustees
3. **investment-one-logo.png** - Investment One
4. **redan-logo.png** - REDAN (Real Estate Developers Association of Nigeria)
5. **airbnb-logo.png** - Airbnb
6. **mbava-logo.png** - Mbava

## Logo Specifications

- **Format**: PNG with transparent background (preferred) or white background
- **Dimensions**: Recommended 240x120px (2:1 aspect ratio)
- **File Size**: Keep under 100KB for optimal performance
- **Style**: 
  - High resolution for retina displays
  - Clean, professional appearance
  - Brand colors preserved

## Notes

- If a logo file is missing, the component will display the partner name as text
- Logos are displayed in grayscale by default and become colored on hover
- Ensure you have proper licensing/permission to use these logos
